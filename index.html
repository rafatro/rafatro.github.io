Desmos Graphing Calculator

A website that creates beautiful charts from inputing equations. It allows not only simple second degree equations like this , but also trigonometric, polar plot, integrals and simple model fitting. Amazing!

Easy CHAID

Web application for creating classification trees using CHAID (Chi-squared Automatic Interaction Detection) algorithm, the same one used by IBM's SPSS software. Easy CHAID was developed by me in Javascript.
