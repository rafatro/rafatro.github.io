Business Analyst channel

YouTube channel in pt-br with videos made by me to teach business analysis for people starting to work in the field or who want to up their game.

Desmos Graphing Calculator

A website that creates beautiful charts from inputting equations. It allows not only simple second-degree equations like this , but also trigonometric, polar plots, integrals, and simple model fitting. Amazing!

Easy CHAID

Web application for creating classification trees using CHAID (Chi-squared Automatic Interaction Detection) algorithm, the same one used by IBM's SPSS software. Easy CHAID was developed by me in Javascript.

CodeCogs Equation Editor

Web application for creating beautiful equation images with transparent background like this from LaTeX markup \overline{x}=\frac{\sum{x_i}}{n}.
